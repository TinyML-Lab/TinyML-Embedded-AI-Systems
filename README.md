# TinyML Embedded AI Systems

This repository contains the final TinyML capstone implementation.

## Project 01: Real-Time Human Activity Recognition

Target: ESP32-S3 + MPU6050

The system reads live IMU data, performs on-device TinyML inference, and prints real-time activity predictions.

## Target Classes

- Idle
- Shake
- Left Tilt
- Right Tilt
- Forward Motion

## Final Output Example

    Prediction: LEFT_TILT
    Confidence: 94.2%
    Inference Time: 22 ms
    RAM Usage: 34 KB
    Flash Usage: 210 KB
