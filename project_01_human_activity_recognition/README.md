# Project 01 - Human Activity Recognition using TinyML

## Objective

Recognize human movements in real time using MPU6050 IMU data and deploy the final model on ESP32-S3.

## Hardware

- ESP32-S3 Dev Board
- MPU6050 6-Axis IMU Sensor

## Pipeline

1. MPU6050 sensor reading
2. Dataset collection
3. Preprocessing
4. Feature engineering
5. Feature selection
6. Model training
7. Quantization
8. ESP32-S3 deployment
9. Live demo
